<a id="user-content-trnum2string" class="anchor" href="#trnum2string" aria-hidden="true"><span class="octicon octicon-link"></span></a>TRnum2string</h1>

<p>It returns the pronunciation of the given integer number in Turkish. Currently it only supports positive integers. I hope to extend it for floating, negative, exponential and ordered numbers. Please note that the output is unicode!</p>
